# Movie Collection Summary

![Sample Movie Collection View](<../../.gitbook/assets/image (2).png>)

This looks at all movies that are indicated to be part of a series or collection of movies. For each collection it gets the other movies in the collection and identified whether they are in the library.

You can:

* Filter records using Checkboxes at the top
* Filter and sort using right-click on Column Headers
* Right-click to add the movie to the library (among other actions)
